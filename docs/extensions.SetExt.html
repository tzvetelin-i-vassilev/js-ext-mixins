Set extension

Methods

filter(callback) → {Set}

Creates a new, filtered down to just the elements from the given set that pass the test implemented by the provided function

Parameters:
Name Type Description
callback function

Predicate, to test each element of the set. Return a value that coerces to true to keep the element, or to false otherwise.
Returns:

A new set filtered down to just the elements from the given set that pass the test implemented by the provided function. If no elements pass the test, an empty set will be returned.

Type
Set

map(callback) → {Set}

Creates a new set populated with the results of calling a provided function on every element in the calling set.

Parameters:
Name Type Description
callback function

Called for every element of set. Each time callback executes, the returned value is added to new set.
Returns:

A new set with each element being the result of the callback function.
