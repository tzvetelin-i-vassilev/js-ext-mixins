# js-ext-mixins

The Javascript extension. Provides additional functionality to classes when they are available and when particular functionality is missing.

## Installation

```shell
npm install --save js-ext-mixins
```

## Importing

```js
import "js-ext-mixins"
```

## Affected classes

 * ArrayBuffer
 * Array
 * CSSStyleSheet
 * DOMMatrix
 * DOMPoint
 * DOMRect
 * Date
 * Function
 * HTMLDocument
 * HTMLElement
 * HTMLImageElement
 * Image
 * Location
 * Number
 * Object
 * Screen
 * Set
 * ShadowRoot
 * SharedArrayBuffer
 * String
 * TypedArray

Functionallity extensions through not existant classes:
 * DOMSize
